Setup in seconds

No configuration files. No messing with /etc/hosts.

1

Register your app

Navigate to your project folder and tell PortPilot to watch it.

~/my-app $ portpilot add
2

Start the server

Run your usual start command through PortPilot or let it auto-detect.

~/my-app $ portpilot start
3

Visit the URL

Your app is now live at a custom local domain securely.

lock my-app.test
